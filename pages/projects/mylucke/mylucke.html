Product

MyLucke is a systems that solves the parking problem. It allows for parkers and parking space owners find spaces and rent with confidence.

Project

The project consisted of mantence, production and design of web and mobile application assets. This included the user interface design for landing pages, and the renter web application.

Process

  • Disocovery and assesment of product core features
  • Production of high-fidelity city landing pages
  • Iteration of web application onboarding wizard design
  • Blackbox quality assurance testing of web and mobile application

Result

The project resulted in the successful iterations of landing pages, the design and ownership of renter onboarding wizard.
